# Smart Drain Monitoring and Number Guessing Project

## 🧩 Overview
This repository combines two projects:
- **Smart Drain Monitoring and Blockage Detection System** – detects and monitors drain blockages using sensors.
- **Number Guessing Game** – a simple Java game where the user guesses a randomly generated number.

## 🚀 Features
- Real‑time drain blockage detection.
- Interactive number guessing logic.
- Modular code structure for easy maintenance.

## 🛠️ Technologies Used
- Java
- Arduino / IoT sensors (for Smart Drain Monitoring)
- Git & GitHub for version control
